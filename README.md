# EduBridge - SDG 4 Quality Education

EduBridge is a study resource sharing platform built to support SDG 4 (Quality Education).

## Features

- User Authentication
- Study Resources
- Student Dashboard
- Opportunities Section
- FastAPI Backend
- React Frontend
- MongoDB Database

## Tech Stack

Frontend:
- React.js
- Tailwind CSS

Backend:
- FastAPI
- Python

Database:
- MongoDB

## SDG Goal

SDG 4 - Quality Education

Ensuring inclusive and equitable quality education and promoting lifelong learning opportunities for all.
